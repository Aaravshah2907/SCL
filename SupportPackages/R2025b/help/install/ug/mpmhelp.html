mpm --help

(Linux, Windows, Mac) Get help using MATLAB Package Manager

collapse all in page

Using mpm --help at the operating system command line requires MATLAB® Package Manager. See Get MATLAB Package Manager.

If you instead want to manage packages within MATLAB, see Package Management (MATLAB).

Syntax

mpm --help

Description

mpm --help or mpm -h displays help information at your operating system command-line for using MATLAB Package Manager. This information includes the operations mpm can perform and its configuration options.

example

Examples

collapse all

Linux® or Mac:

./mpm --help

Windows® (run as administrator):

.\mpm.exe --help
