mpm --version

(Linux, Windows, Mac) Get MATLAB Package Manager version information

collapse all in page

Using mpm --version at the operating system command line requires MATLAB® Package Manager. See Get MATLAB Package Manager.

If you instead want to manage packages within MATLAB, see Package Management (MATLAB).

Syntax

mpm --version

Description

mpm --version or mpm -v displays the version of MATLAB Package Manager you are using and the platform it runs on.

example

Examples

collapse all

Linux® or Mac:

./mpm --version

Windows® (run as administrator):

.\mpm.exe --version
