Get New MATLAB Release

When a new MATLAB® release becomes available, you can download and install the new release and, optionally, uninstall any previous releases.

Prerequisites

  • Your MATLAB license must be current on Software Maintenance Service. For additional information, see MathWorks Software Maintenance Service.

  • If you are switching from prerelease to a new release, you must uninstall the prerelease version first. See Uninstall MATLAB.

Check for New Release

When working in MATLAB, you can check to see if a new release is available. In the Resources section of the Home tab, click Help and select Check for Updates. The Add-On Manager window opens with the Updates tab selected. If a new release is available, the window displays a banner prompting you to get the new release.

Get New Release

To get a new MATLAB release, follow these steps:

  1. From MathWorks Downloads, download the installer for the new release.

  2. Run the installer and follow the installation prompts. Keep these points in mind:

    • The product selection list displays only the products currently on your license. For example, if your existing installation includes ten products, but your current license includes only nine of those products, the tenth product is not available to install.

    • You must install the new release in a new installation folder. You cannot install the release in a folder that contains a previous MATLAB installation.

After getting the new release, you can optionally uninstall any previously installed MATLAB releases that you are no longer using.

Get Previously Installed Support Packages

If you installed support packages for a previous MATLAB release, the software notifies you that you can install these support packages for your new release. To do so, click Install from the notification and follow the prompts for installation. Alternatively, at the MATLAB command prompt, enter this command:

upgradePreviouslyInstalledSupportPackages
