mpmListRepositories

List known repositories

Since R2024a

collapse all in page

    Syntax

    mpmListRepositories
    repo = mpmListRepositories

    Description

    mpmListRepositories displays the names and locations of repositories on the MATLAB repository list.

    example

    repo = mpmListRepositories returns the repository list as an array of matlab.mpm.Repository objects.

    Examples

    collapse all

    Display a list of the known repositories.

    mpmListRepositories
     mpmListRepositories
        Name                    Location            
    _____________    _______________________________

    "SharedRepo"     "M:\SharedCode\SharedRepository"       
    "MyRepo"         "C:\MyCode\MyPackageRepository"
    "DepartmentRepo" "Z:\Astro\PackageRepo"
