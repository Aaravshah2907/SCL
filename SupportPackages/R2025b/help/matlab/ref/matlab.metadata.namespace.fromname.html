matlab.metadata.Namespace.fromName

Class: matlab.metadata.Namespace
Namespace: matlab.metadata

Return matlab.metadata.Namespace object for specified namespace

Renamed from meta.package.fromName in R2024a

expand all in page

Syntax

mnsp = matlab.metadata.Namespace.fromName(namespaceName)

Description

mnsp = matlab.metadata.Namespace.fromName(namespaceName) returns the matlab.metadata.Namespace object associated with the specified namespace. If namespaceName is a nested namespace, then you must provide the fully qualified name (for example, nsp1.nsp2).

example

Input Arguments

expand all

Namespace name, specified as a character vector or string scalar.

Attributes

Statictrue

To learn about attributes of methods, see Method Attributes.

Examples

Get Namespace Contents

Return the names of the classes contained in the event namespace.

mev = matlab.metadata.Namespace.fromName("event");
for k=1:length(mev.ClassList)
   disp(mev.ClassList(k).Name)
end
event.EventData
event.ClassInstanceEvent
event.PropertyEvent
event.ParentEvent
event.ChildEvent
event.DynamicPropertyEvent
event.listener
event.proplistener

Version History

Introduced in R2008a

expand all

The meta.package.fromName method has a new namespace and class name, matlab.metadata.Namespace.fromName. The behavior will remain the same, and MATLAB® will continue to recognize the old name.
