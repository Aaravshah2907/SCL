propertyeditor

(Removed) Show or hide the Property Editor

collapse all in page

    propertyeditor has been removed. Use inspect to launch the Property Inspector instead.

    Syntax

    propertyeditor
    propertyeditor(state)
    propertyeditor(fig,___)

    Description

    propertyeditor shows the Property Editor plot tool on the current figure. If no current figure exists, propertyeditor creates one.

    example

    propertyeditor(state) changes the state of the Property Editor plot tool on the figure.

    propertyeditor(fig,___) operates on the specified figure instead of the current figure. The input fig can precede any of the input argument combinations in the previous syntaxes.

    Examples

    collapse all

    Show the Property Editor on a figure by calling the propertyeditor function. The tool opens in the same configuration as last time you invoked the tool.

    plot(1:10);
propertyeditor

    Input Arguments

    collapse all

    Property Editor tool state, specified as one of these values:

    • "on" — Show Property Editor.

    • "off" — Hide Property Editor.

    • "toggle" — Switch the Property Editor state between the "on" and "off" states.

    Target figure, specified as a Figure object. Use fig to show or hide the Property Editor on a specific figure instead of the current figure.

    Version History

    Introduced before R2006a

    expand all

    The propertyeditor function has been removed. Instead, to interactively modify graphics object properties, use inspect to launch the Property Inspector.
