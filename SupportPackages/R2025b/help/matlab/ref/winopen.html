winopen

Open file in appropriate application (Windows)

collapse all in page

Syntax

winopen name

Description

winopen name opens the specified file in the Microsoft® Windows® application associated with the extension of the file. winopen performs the same action as double-clicking the file in Windows Explorer.

example

Examples

collapse all

Open Script

Open the file thesis.docx in Microsoft® Word.

winopen thesis.docx
Open Script

Open the file my_sine_wave.html in the system Web browser.

winopen my_sine_wave.html
Open Script
winopen(cd)

Input Arguments

collapse all

Name of file to open, specified as a character vector. name can be an absolute or relative path.
