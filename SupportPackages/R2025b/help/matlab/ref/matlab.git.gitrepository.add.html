add

Mark new file for addition to Git repository

Since R2023b

collapse all in page

    Syntax

    add(repo,files)

    Description

    add(repo,files) marks the files files for addition to the Git™ repository repo.

    example

    Examples

    collapse all

    Create a new Git repository and a repository object.

    repo = gitinit("newrepo");

    Create a new script.

    edit newScript.m;

    Mark the script file for addition to the repository.

    add(repo,"newScript.m");

    Commit the new file to the repository.

    commitDetails = commit(repo,message="Add file")

    Input Arguments

    collapse all

    Git repository, specified as a matlab.git.GitRepository object.

    Files to add to the Git repository, specified as a string array, character vector, or cell array of character vectors.

    Example: "newfile.m"

    Data Types: char | string | cell

    Tips

    When you perform a merge, you can use the add or rm functions to mark the conflict on a file as resolved.
