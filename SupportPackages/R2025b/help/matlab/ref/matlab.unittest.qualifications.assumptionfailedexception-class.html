matlab.unittest.qualifications.AssumptionFailedException Class

Namespace: matlab.unittest.qualifications
Superclasses: MException

Exception used for assumption failures

Description

The matlab.unittest.qualifications.AssumptionFailedException class defines an exception that is thrown when an assumption failure occurs.

The testing framework instantiates this class. You cannot create an object of the matlab.unittest.qualifications.AssumptionFailedException class directly.

Version History

Introduced in R2013a
