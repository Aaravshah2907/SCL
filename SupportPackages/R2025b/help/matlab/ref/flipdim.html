flipdim

(Not recommended) Flip array along specified dimension

flipdim is not recommended. Use flip instead.

Syntax

B = flipdim(A,dim)

Description

B = flipdim(A,dim) returns A with dimension dim flipped.

When the value of dim is 1, the array is flipped row-wise down. When dim is 2, the array is flipped columnwise left to right. flipdim(A,1) is the same as flipud(A), and flipdim(A,2) is the same as fliplr(A).

Examples

flipdim(A,1) where

A =

     1     4
     2     5
     3     6

produces

     3     6
     2     5
     1     4

Extended Capabilities

expand all

The flipdim function supports tall arrays with the following usage notes and limitations:

  • If specified, the dim argument must be greater than 1.

For more information, see Tall Arrays.

Usage notes and limitations:

  • Does not support cell arrays for the first argument.

Introduced before R2006a
