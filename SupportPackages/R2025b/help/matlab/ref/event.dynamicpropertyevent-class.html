event.DynamicPropertyEvent Class

Namespace: event
Superclasses: event.EventData

Event data for dynamic property events

expand all in page

Description

The event.DynamicPropertyEvent class defines the event data passed to listeners of the dynamicprops PropertyAdded and PropertyRemoved events.

event.DynamicPropertyEvent is a subclass of event.EventData. The event.DynamicPropertyEvent class is sealed (cannot be subclassed) and its constructor is private.
