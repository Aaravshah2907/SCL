Insert String Set Property

The String Set dialog box enables you to define a list of valid string values for a property.

  • Name — Property name. You can use any valid MATLAB® property name.

  • Value — Valid string for property. Each value must be unique.

  • Default — Indicates the default value for this property.

  • Insert — Create the string set and add the property with the default value to your code.

  • Add or remove property values with the + (plus) and - (minus) controls, respectively.

When you click Insert, the MATLAB Editor inserts the default property value and a PropertySet property into your code. In this example, the property name is Color, and the property values are red, green, and blue. The default is red. The MATLAB Editor inserts the following code into the System object™ definition:

properties (Nontunable)
  Color = 'red';
end

properties(Constant, Hidden)
  ColorSet = matlab.system.StringSet({'red','green','blue'});
end
