readFrame

Return oldest image frame from board, web, or ArduCam module camera

collapse all in page

Add-On Required: This feature requires the MATLAB Support Package for Raspberry Pi Hardware add-on.

Syntax

data = readFrame(mycamera)
[data,ts] = readFrame(mycamera)
data = readFrame(mycamera,count)
[data,ts] = readFrame(mycamera,count)
data = readFrame(wcam)
[data,ts] = readFrame(wcam)
data = readFrame(wcam,count)
[data,ts] = readFrame(wcam,count)
data = readFrame(aCamObj)
[data,ts] = readFrame(aCamObj)
data = readFrame(aCamObj,count)
[data,ts] = readFrame(aCamObj,count)

Description

data = readFrame(mycamera) returns the oldest RGB image frame, as obtained from the Raspberry Pi® camera board, as a 3-D image.

[data,ts] = readFrame(mycamera) also returns the timestamp of the oldest image.

data = readFrame(mycamera,count) returns multiple RGB image frames, as obtained from the Raspberry Pi camera board, as a 4-D array. The count value specifies the number of frames, with a maximum allowed value of 10. The oldest image is returned first. If count = 1, the function returns a 3-D image.

[data,ts] = readFrame(mycamera,count) also returns the timestamps corresponding to each image frame.

data = readFrame(wcam) returns the oldest RGB image frame, as obtained from the web camera attached to the Raspberry Pi board, as a 3-D image.

[data,ts] = readFrame(wcam) also returns the timestamp of the oldest image.

data = readFrame(wcam,count) returns multiple RGB image frames, as obtained from the web camera attached to the Raspberry Pi board, as a 4-D array. The count value specifies the number of frames, with a maximum allowed value of 10. The oldest image is returned first. If count = 1, the function returns a 3-D image.

[data,ts] = readFrame(wcam,count) also returns the timestamps corresponding to each image frame.

data = readFrame(aCamObj) returns the oldest RGB image frame from the selected camera on ArduCam Multi Camera Adapter Module as a 3-D array.

[data,ts] = readFrame(aCamObj) also returns the timestamp of the oldest image.

data = readFrame(aCamObj,count) returns multiple RGB image frames, as obtained from the selected camera attached to the ArduCam Multi Camera Adapter Module, as a 4-D array. The count value specifies the number of frames, with a maximum allowed value of 10. The oldest image is returned first. If count = 1, the function returns a 3-D image.

[data,ts] = readFrame(aCamObj,count) also returns the timestamps corresponding to each image frame.

mypi = raspi;

Input Arguments

collapse all

Connection to camera board on the Raspberry Pi hardware, specified as a cameraboard object.

Example: mycamera

Connection to a USB web camera connected to Raspberry Pi board, specified as a webcam object.

Example: wcam

Connection to a camera on the ArduCam Multi Camera Adapter Module, specified as an arducam object.

Example: aCamObj

Number of frames to be considered for obtaining the 4-D array of RGB images. The maximum value of count is 10.

Data Types: double

Output Arguments

collapse all

4-D array of RGB images, returned as RGB-by-n matrix of values, where n is the value specified as count. If count = 1, data is the 3-D image frame.

Data Types: uint8

Timestamp of the RGB images, returned as N-by-1 matrix, where n is the value specified in count.

Extended Capabilities

expand all

  • You can also deploy readFrame on the Raspberry Pi hardware in MATLAB Online™.

  • Bullseye and Bookworm OS only supports C++ code generation while using readFrame function with cameraboard object.
