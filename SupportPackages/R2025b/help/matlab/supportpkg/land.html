land

Land Ryze drone

collapse all in page

Add-On Required: This feature requires the MATLAB Support Package for Ryze Tello Drones add-on.

Syntax

land(droneObj)

Description

land(droneObj) initiates the gradual landing of Ryze drone, represented by droneObj, from the current position and ends the drone flight. Use takeoff to begin a new flight.

example

Examples

collapse all

Connect to a Ryze Tello drone.

droneObj = ryze()
droneObj = 
          ryze with properties:

                    Name: "Tello"
                      ID: "TELLO-D2B07B"
               IPAddress: "192.168.10.1"
                   State: "landed"
            BatteryLevel: 50%
             StationMode: 0
        AvailableCameras: ["FPV"]

Use takeoff to initiate takeoff of the drone.

takeoff(droneObj)

While the Ryze Tello drone is in flight, initiate landing of the drone.

land(droneObj)

Input Arguments

collapse all

Ryze drone connection object, specified as a droneObj.
