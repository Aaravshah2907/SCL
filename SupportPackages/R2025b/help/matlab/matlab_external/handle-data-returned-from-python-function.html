Automatically Convert Python Types to MATLAB Types

MATLAB® automatically converts these data types returned from Python® functions to MATLAB types.

In addition, when a Python dictionary or a Python Pandas DataFrame containing a Python string object is returned from Python to MATLAB, the Python string object is converted to a MATLAB string.

If the output of a Python function has a type, such as numpy.ndarray, that implements the Python buffer protocol, then MATLAB displays:

  • The actual Python type

  • The underlying data

  • The corresponding MATLAB conversion function. Use this function to fully convert the Python object to a MATLAB array.

For example, a Python function returns this array p:

p = 

  Python ndarray:

     8     1     6
     3     5     7
     4     9     2

    Use details function to view the properties of the Python object.

    Use double function to convert to a MATLAB array.

Convert p to a MATLAB matrix P by using the suggested conversion function.

P = double(p)
