Current Folder Settings

You can specify the number of files that display in the Files panel, and customize their appearance.

On the Home tab, in the Environment section, click Settings. Then, select MATLAB > Current Folder.

SettingUsage
History

Specify the number of recently used folders maintained in the Current Folder toolbar drop-down list using the Number of most recent folders to save setting.

Path indication

Select the Indicate inaccessible files check box to dim the display of files and folders inaccessible to MATLAB®.

Hidden Files

Select the Show hidden files and folders check box to display files and folders in the Files panel that the operating system hides from system file browsers and file-listing commands.

This setting does not apply to Microsoft® Windows® platforms. On Windows platforms, the Files panel follows the Windows setting for showing hidden files. To set or change the Windows setting, access the Folder Options, and then select an option for viewing Hidden files and folders.

To specify the current folder in MATLAB at startup, use the General Settings.

To change how dates display in the Files panel, change the short date format for your operating system. MATLAB uses your operating system's short date format to display dates in both the Files panel and the Command History.
