matlab::data::Reference<TypedArray<T>>

Templated C++ class to get reference to TypedArray

Description

The Reference<TypedArray<T>> class extends the APIs available to a reference to an Array. It derives from the Reference<Array> class and provides iterators and type-safe indexing. Use to modify the elements of a container such as a struct or cell.

Reference<TypedArray<T>> is not thread-safe - do not pass references to TypedArray objects between threads.

TypedArrayRef is defined in TypedArrayRef.hpp as:

template <typename T>
using TypedArrayRef = Reference<TypedArray<T>>;

Parameters

First array index.

Returns

Temporary object containing index specified. If type T is const, then the return value allows the element of the array to be retrieved, but not modified. Otherwise, the element can be modified or retrieved.
