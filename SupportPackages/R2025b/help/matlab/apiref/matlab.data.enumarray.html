matlab::data::EnumArray

C++ class to access MATLAB enumeration arrays

Description

Use EnumArray objects to access enumeration arrays. To create an EnumArray, call createEnumArray in the ArrayFactory class.

Examples

Display enum Value

Define an enumeration class:

classdef MyClass
   enumeration
       A
       B
       C
   end
end

Define a matlab::data::EnumArray object for the MyClass.C enumeration argument and display the value.

#include "MatlabDataArray.hpp"
#include "MatlabEngine.hpp"
#include <iostream>

int main()
{
	using namespace matlab::data;
	ArrayFactory f;
	auto e = f.createEnumArray({ 1,1 }, "MyClass", { "C" });
	// Display enum value
	std::cout << "Property value: " << std::string(e[0]) << std::endl;
	return 0;
}
